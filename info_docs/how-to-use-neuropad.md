# How To Use NeuroPad

This guide focuses on the day-to-day workflow inside the app.

## Create a notebook

1. Launch the desktop app.
2. Enter a title in the notebook title field.
3. Enter a target file path (example: `C:\notebooks\my_work.npad`).
4. Click `New`.

## Add and edit cells

- `+ Markdown`: Adds a markdown cell for notes.
- `+ Go Cell`: Adds a Go code cell.
- `+ Ruby Cell`: Adds a Ruby code cell.
- Type directly in the cell text area.

## Run code cells

1. Find a code cell.
2. Click `Run`.
3. Inspect outputs shown under the cell.

## Save and open notebooks

- `Save`: Writes the current notebook to the given path.
- `Open`: Loads a notebook from the given path.

Tip: keep the file path field accurate before using `Open` or `Save`.
